Сметчик PRO 5.2
            Программа для составления смет с поддержкой двух вариантов цены

            📋 ОСНОВНЫЕ ВОЗМОЖНОСТИ
            1. Составление сметы
               • Добавление работ из справочника с указанием объёма
               • Автоматический расчёт материалов по нормам расхода
               • Два варианта цены с автоматическим расчётом стоимости
               • Сравнение вариантов и расчёт экономии
               • Учёт доп. расходов: накладные, транспортные, подъёмные механизмы, вывоз мусора
            2. Справочник работ и материалов
               • Хранение работ с ценами и единицами измерения
               • Привязка материалов к работам с нормами расхода
               • Два варианта цены для каждой позиции
               • Добавление, редактирование, удаление записей
               • Поиск и фильтрация по названию работы или материала
               • Сортировка по любому столбцу
            3. Импорт и экспорт
               • Выгрузка сметы в Excel с формулами и форматированием
               • Автоматическое сохранение метаданных (Meta) для восстановления
               • Загрузка ранее экспортированных смет\n
               • Автоматическое добавление новых записей в справочник при импорте
            4. Черновик сметы
               • Автосохранение при изменении сметы
               • Восстановление черновика при запуске программы
               • Ручное сохранение кнопкой «💾 Черновик»

            ⌨️ ГОРЯЧИЕ КЛАВИШИ
               Ctrl+C / Ctrl+V / Ctrl+X — копировать / вставить / вырезать
               Ctrl+Z — отмена последнего изменения в смете
               ПКМ — контекстное меню (копировать/вставить)

            📊 ДВА ВАРИАНТА ЦЕНЫ
               Вариант 1 (В1) и Вариант 2 (В2) — независимые цены
               Автоматический расчёт В1 от В2 по коэффициенту
               Расчёт экономии между вариантами в процентах

            💾 ХРАНИЛИЩЕ ДАННЫХ
               • База данных: SQLite (.db) — нормализованная структура
               • Поддержка Excel (.xlsx) для совместимости
               • Автоматическая миграция из Excel в SQLite

            🔧 НАСТРОЙКИ
               • Папка для базы данных
               • Папка для сохранения смет
               • Коэффициент автоподстановки цены В1 от В2 для материалов
               • Коэффициент автоподстановки цены В1 от В2 для работ
